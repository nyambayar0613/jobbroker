■ 네이버 블로그 API 연동 설정 (기사 송고 방법)
1. 네이버 계정에 가입하신 후 블로그를 생성합니다.
(기존 블로그 사용자는 내 블로그의 관리자 설정 페이지에서 API등록이 가능합니다.)
2. 내 블로그에서 관리자 설정 페이지로 들어갑니다.
3. 환경설정에서 [글쓰기 API설정]을 클릭합니다.
4. 글쓰기 API설정에서 [암호발급]버튼을 눌러 API연결 암호를 생성합니다.
5. 뉴스 관리자 API 설정
① 사용여부 : 기사를 외부 송고하실 경우 사용란에 체크합니다.
② 기본 카테고리 : 송고하실 기사가 등록될 네이버 블로그의 카테고리명을 입력합니다.
※ 카테고리 생성 : 카테고리명은 블로그의 관리자 설정 - 메뉴관리 - 블로그에서 생성합니다.
③ 블로그 정보 추가 : 추가 등록하실 블로그의 API정보를 입력할 수 있습니다.
④ 블로그 API URL : 블로그 글쓰기 API 설정에서 API연결 UIL을 입력합니다.
⑤ 블로그 아이디 : 블로그 아이디를 입력합니다.
⑥ API 연결암호 : 블로그 API연결정보에서 API연결 암호를 입력합니다.
