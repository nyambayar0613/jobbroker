■ 네이버 신디케이션 설정 방법
신디케이션이란?
신디케이션이란 웹 사이트 또는 모바일 어플리케이션(이하 앱 이라고 한다)이 보유하고 있는 콘텐츠를 다른 웹 사이트가 이용할 수 있게 하는 방식의 하나입니다. 콘텐츠를 보유하고 있는 웹 사이트와 앱은 정해진 형식에 따라 문서를 구성하여 해당 문서의 링크를 다른 웹 사이트에 전송하고, 이를 전송 받은 웹 사이트는 해당 문서의 내용을 분석하여 사용합니다.
1. 네이버 웹마스터도구 (http://webmastertool.naver.com/)로 이동하여 네이버 회원가입 후 로그인 합니다.
(기존 네이버 사용자는 로그인 합니다.)
2. 웹마스터도구의 연동할 사이트 추가 버튼을 클릭합니다.
3. 연동할 사이트의 주소를 입력합니다.
4. 사이트 소유 미확인으로 등록될 경우 소유확인 버튼을 클릭합니다.
5. 사이트소유 확인 방법증 HTML 태그를 선택한 후 메타태그를 복사하여 관리자의 메타테그 입력란에 붙여넣습니다.
6. 사용여부가 확인되면 화면에 등록된 사이트 주소를 클릭합니다.
7. 웹마스터도구 설정에서 신디케이션 인증키(토큰)을 복사하여 관리자의 토큰 입력란에 붙여 넣습니다.
