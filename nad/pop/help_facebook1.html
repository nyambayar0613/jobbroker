■ 페이스북 앱 ID 발급 방법
페이스북 앱 ID란?
페이스북 앱을 개발하거나 사용하기 위해서는 앱을 페이스북에 등록하여 앱 아이디와 토큰을 발급받아야 합니다.
1. 페이스북 개발자 사이트 (https://developers.facebook.com)로 이동하여 페이스북 계정으로 로그인 합니다. 계정이 없으시면 상단 로그인 버튼을 눌러 페이스북 계정을 만듭니다.
2. 개발자 페이지 상단에 있는 My Apps의 Add a New App을 클릭합니다.
3. 다음과 같은 팝업 창이 뜨면 www(웹사이트)를 선택합니다.
4. 텍스트 입력란에 등록할 앱의 이름을 입력합니다.
5. 하단 메뉴에서 생성할 앱의 카테고리를 선택한 후 앱 ID만들기 버튼을 클릭합니다.
6. 화면 하단 입력란에 사이트 주소를 입력하고 Next버튼을 클릭합니다.
7. 아래와 같은 화면이 나타나면 앱 등록이 완료됩니다.
8. 다시 화면 상단 My App의 See All Apps를 클릭하여 등록된 앱을 확인합니다.
9. 비활성화된 앱을 클릭합니다.
10. 화면 죄측 상단의 Settings 버튼을 클릭하여 App 기본 정보(Display Name, Namespace, App Domains, Email)를 입력한 후 Save Changes 버튼을 눌러 입력 정보를 저장합니다.
11. 다시 죄측 메뉴 상단의 Dashboard 버튼을 클릭하여 App Id 및 App Secret 코드를 확인합니다. (App Scret 코드는 Show 버튼을 클릭하여 확인할 수 있습니다.)
12. 뉴스 홈페이지 관리자의 환경설정 - 사이트관리 - 기본정보설정에서 페이스북 앱 ID / 페이스북 앱 Scret 코드를 입력한 후 저장버튼을 눌러 앱 연동을 완료합니다.
