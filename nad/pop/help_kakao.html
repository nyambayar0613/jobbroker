■ 카카오 API 키 발급 방법
1. 카카오 개발자 사이트 (https://developers.kakao.com)로 이동하여 카카오 계정으로 로그인 합니다.
2. 앱만들기 페이지에서 이름 입력란에 앱명을 입력하고 Create버튼을 클릭합니다.
3. 등록이 완료되면 화면 하단에 [설정]을 클릭합니다.
4. 내애플리케이션 기본정보 하단에 있는 플랫폼 추가 버튼을 클릭합니다.
5. 팝업창이 뜨면 플랫폼 추가 방식을 웹으로 체크하신 후 등록하실 사이트의 도메인 주소를 입력하고 추가하기 버튼을 클릭합니다.
6. 플랫폼이 추가되면 상단의 키값 중 JavaScript 키를 복사합니다.
7. 홈페이지 관리자의 환경설정 - 사이트관리 - 기본정보설정에서 카카오 API 키를 입력한 후 저장버튼을 눌러 앱 연동을 완료합니다.
