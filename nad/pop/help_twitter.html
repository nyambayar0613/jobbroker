■ 트위터 컨슈머 Key 와 Screct 발급 방법
트위터의 컨슈머키와 컨슈머 시크릿이 있어야 리턴받는 페이지를 통해 리퀘스트 토큰을 얻어 정보나 서비스를 끌어올 수 있습니다.
1. Twitter Apps (https://apps.twitter.com/)로 이동하여 트위터 계정으로 로그인 합니다.
2. Creat New App 버튼을 눌러 트위터 앱을 생성합니다.
3. 신규 앱 등록을 위한 정보 입력란에 앱이름, 설명, 사이트주소, Callback URL을 입력하고 확인란을 체크하신 후 Create your Twitter application 버튼을 클릭합니다. 이때 callback URL 주소는 http://등록할 사이트 URL 주소/member/process/twitter_callback.php 입니다. ( 예시 - http://news2.netfu.co.kr/member/process/twitter_callback.php )
만약 신규 앱 생성시 오류가 발생하면 트위터 계정에서 휴대폰 등록 여부 등을 확인해 보시기 바랍니다.
4. 등록된 앱은 Keys and Access Tokens에서 컨슈머 키 와 컨슈머 시크릿을 확인 하실 수 있습니다.
5. 뉴스 홈페이지 관리자의 환경설정 - 사이트관리 - 기본정보설정에서 트위터 컨슈머 Key / 트위터 컨슈머 Secret 코드를 입력한 후 저장버튼을 눌러 앱 연동을 완료합니다.
