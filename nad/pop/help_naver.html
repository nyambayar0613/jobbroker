■ 네이버 클라이언트 ID / Secret 발급 방법
네이버 클라이언트 인증
네이버 클라이언트 인증 서비스에 등록하시면 네이버에서 제공하는 다양한 OpenAPI를 이용하실 수 있습니다.
1. Naver Login Developers (https://nid.naver.com/devcenter/register.nhn)로 이동하여 네이버 계정으로 로그인 하신 후 메뉴에서 애플리케이션 등록을 선택합니다.
2. ① 새 애플리케이션 등록에서 애플리케이션 이름을 입력하고 애플리케이션에 맞는 카테고리를 선택한 후 서비스 환경을 web으로 체크합니다.
② PC웹 및 Mobile웹에 맞는 서비스 URL(사이트 주소) 및 Callback URL을 입력합니다.
[PC웹 callback URL 주소]
 http://등록할 사이트 URL 주소/member/process/naver_callback.php
http://등록할 사이트 URL 주소/member/process/naver_redirect.php
※ 오른쪽 +아이콘을 클릭하면 주소를 추가하실 수 있습니다.
[Mobile웹 callback URL 주소]
http://등록할 사이트 모바일 URL 주소/m/member/process/naver_callback.php
③ 이용약관에 동의하고 확인버튼을 누르면 애플리케이션 등록이 완료됩니다.
3. 등록된 내 애플리케이션의 메뉴에서개요를 선택하여 Client ID / Client Secret 을 확인 후 키 값을 복사합니다.
4. 뉴스 홈페이지 관리자의 환경설정 - 사이트관리 - 기본정보설정에서 네이버 Client ID / 네이버 Client Secret를 입력한 후 저장버튼을 눌러 앱 연동을 완료합니다.
